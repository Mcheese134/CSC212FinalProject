Setup Instructions:

1. Pip or Pip3 install Flask and Python3

2. Flask Instruction: https://realpython.com/flask-by-example-part-1-project-setup/

3. Once you get to "source env/bin/activate", pip install the necessary libraries

4. pip install -r requirements.txt in the main directory

6. Finally, do python3 app.py to execute the application.

6. You can find the result in: http://127.0.0.1:5000/
